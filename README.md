# Анализ данных Яндекс Афиши

## Описание проекта
В данном проекте проводится анализ данных о пользователях платформы Яндекс Афиша. Цель исследования — определить факторы, влияющие на возвращаемость пользователей.

## Используемые библиотеки
- pandas
- matplotlib
- seaborn
- phik
- sqlalchemy

## Структура проекта
- `.gitignore`: Исключает временные файлы.
- `requirements.txt`: Список зависимостей.
- `README.md`: Описание проекта.
- `.env`: Хранит конфиденциальные данные (не попадает в репозиторий).
